# 🎮 Dashboard de Vendas - Microsoft Xbox

## 📋 Sobre o Projeto
Este repositório contém um dashboard estratégico de vendas desenvolvido em Excel, focado na análise de desempenho de produtos da linha Xbox. O objetivo é facilitar a tomada de decisão através da visualização de KPIs críticos.

## 📊 Funcionalidades e Análises
- **KPIs Principais:** Visualização rápida de faturamento total e volume de vendas.
- **Análise Temporal:** Gráficos que mostram a evolução das vendas ao longo dos meses.
- **Segmentação:** Filtros interativos para análise por categoria e região.

## 🛠️ Tecnologias Utilizadas
- **Microsoft Excel Cloud**
- **Tabelas Dinâmicas**
- **Segmentadores de Dados (Slicers)**

## 📂 Arquivos no Repositório
- `base.xlsx`: Dados brutos utilizados para a construção.
- `Desafio_Dashboard_Vendas_Xbox.xlsx`: Arquivo final com o dashboard interativo.

## 🚀 Como visualizar
1. Faça o download do arquivo `.xlsx`.
2. Abra no Microsoft Excel (Desktop ou Web).
3. Utilize os botões de segmentação para interagir com os gráficos.
